# strsx.com

Strsx is my interactive e-portfolio featuring a dark hacker/cyberpunk aesthetic, a fully functional terminal UI, and an AI-powered chatbot. 

Feel free to reuse my website in this repository.

## Design Guidelines

For details on the color palette, typography, components, and the overall styling standards of the website, please refer to the [DESIGN.md](DESIGN.md) document.

## Features
- **Interactive Terminal UI**: A hidden challenge console for an engaging user experience.
- **AI Chatbot**: A custom-styled floating chatbot that integrates with a backend API to answer questions about skills and experience.
- **Glassmorphism & Neon Accents**: Sleek UI design using modern CSS capabilities.

## Usage

To start development:
```bash
npm run serve
```

To build the project:
```bash
npm run build
```

[Visit strsx.com](https://strsx.com/)

## GitHub Webpage Branch
GitHub Pages is used on the `gh-pages` branch. 
Please copy the built output files to that branch for web deployment.
